# DO SPORTS 실시간 시간표 v22

## v22 변경 내용

- 모바일/PC 학부모 화면에서 보기 버튼, 확대 버튼, 요일 이동 버튼 없이 시간표만 보이도록 유지합니다.
- 좌우 스크롤 없이 `시간 | 월 | 화 | 수 | 목 | 금` 전체표를 한 화면 폭에 맞춰 표시합니다.
- 같은 시간대에 수업이 여러 개 있으면 카드가 옆으로 좁아지거나 내부 스크롤이 생기지 않고, 해당 시간대 행 자체가 아래로 커집니다.
- 긴 수업도 시간대를 넘어 길게 이어지지 않고, 시작 시간대 칸 안에서 실제 수업 시간을 카드 안에 표시합니다.
- 카드 텍스트 말줄임과 잘림을 줄이고, 세로로 자연스럽게 늘어나는 방식으로 정리했습니다.
