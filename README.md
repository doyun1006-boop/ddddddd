# DO SPORTS 실시간 시간표 v19

Netlify + GitHub로 배포하는 DO SPORTS 실시간 시간표입니다.

## v19 주요 변경

- 모바일/PC 모두 기본 보기에서 주간 시간표가 전체적으로 들어오도록 `전체보기` 모드를 추가했습니다.
- 시간표 전체를 축소/확대하는 방식으로 변경했습니다.
  - 전체보기: 화면 너비에 맞춰 자동 축소
  - 80%, 100%, 125%, 150%: 원하는 크기로 확대/축소
- 겹치는 수업이 있을 때 글씨를 작게 줄이거나 말줄임 처리하는 대신, 기본 카드 가독성을 유지하고 전체 시간표 크기만 조절합니다.
- `100%`, `125%`, `150%` 모드에서는 좌우 스크롤로 자세히 볼 수 있습니다.
- `전체보기` 모드는 월~금 또는 토~일 전체 흐름을 한눈에 보는 용도입니다.
- 관리자 페이지, 수업분류 추가, 상담/버튼 추가 기능은 기존 v18 기준으로 유지됩니다.
