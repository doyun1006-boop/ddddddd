# DO SPORTS 실시간 시간표 v12

Netlify + GitHub로 배포하는 DO SPORTS 실시간 시간표입니다.

## v12 변경사항

- 학부모 화면에서 `편집` 버튼을 제거했습니다.
- 관리자 화면에서 상단 상담/신청 버튼을 원하는 만큼 추가할 수 있게 변경했습니다.
- 관리자 화면에서 버튼명, 링크 주소, 강조 여부를 직접 수정할 수 있습니다.
- 버튼 순서 변경, 삭제 기능을 추가했습니다.
- 기존 `개설 희망`, `반 모으기`, `상담 및 신청서 작성` 버튼은 자동으로 새 버튼 목록으로 변환됩니다.
- 기존 시간표, 관리자 비밀번호, Netlify Blobs 저장 방식은 그대로 유지됩니다.

## GitHub 업로드 구조

GitHub 저장소 첫 화면에 아래 항목이 바로 보여야 합니다.

```text
netlify
public
README.md
netlify.toml
package.json
```

`academy-live-schedule-v12` 폴더 자체를 올리지 말고, 그 안의 파일과 폴더를 올리세요.

## Netlify 환경변수

기존과 동일하게 아래 3개가 필요합니다.

```text
ADMIN_PASSWORD
NETLIFY_SITE_ID
NETLIFY_BLOBS_TOKEN
```

환경변수를 이미 설정했다면 다시 만들 필요 없습니다. 파일만 업로드하고 재배포하면 됩니다.

## 관리자에서 수정 가능한 항목

- 학원명
- 메인 제목
- 메인 설명
- 공지사항
- 상단 버튼 목록 추가/수정/삭제/순서 변경
- 수업 종목
- 요일
- 시작/종료 시간
- 수업명
- 학년
- 정원
- 현재원
- 장소

## 배포 후 주소

- 학부모 화면: `/`
- 관리자 화면: `/admin`
- 저장 함수 확인: `/.netlify/functions/schedule`
