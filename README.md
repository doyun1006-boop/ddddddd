# DO SPORTS 실시간 시간표 v13

Netlify + GitHub로 배포하는 DO SPORTS 실시간 시간표입니다.

## v13 변경사항

- 관리자 화면에서 `농구 / 축구 / 키즈` 외의 수업분류 탭을 직접 추가할 수 있게 변경했습니다.
- 수업분류 이름과 아이콘을 관리자 화면에서 수정할 수 있습니다.
- 수업분류 순서를 위/아래로 이동할 수 있습니다.
- 수업분류를 삭제할 수 있습니다. 해당 분류에 수업이 있으면 다른 분류로 이동됩니다.
- 학부모 화면의 탭도 관리자에서 만든 수업분류 기준으로 자동 표시됩니다.
- 기존 상담/신청 버튼 추가 기능과 시간표 관리 기능은 그대로 유지됩니다.

## GitHub 업로드 구조

GitHub 저장소 첫 화면에 아래 항목이 바로 보여야 합니다.

```text
netlify
public
README.md
netlify.toml
package.json
```

`academy-live-schedule-v13` 폴더 자체를 올리지 말고, 그 안의 파일과 폴더를 올리세요.

## Netlify 환경변수

기존과 동일하게 아래 3개가 필요합니다.

```text
ADMIN_PASSWORD
NETLIFY_SITE_ID
NETLIFY_BLOBS_TOKEN
```

환경변수를 이미 설정했다면 다시 만들 필요 없습니다. 파일만 업로드하고 재배포하면 됩니다.

## 관리자에서 수정 가능한 항목

- 학원명
- 메인 제목
- 메인 설명
- 공지사항
- 상단 버튼 목록 추가/수정/삭제/순서 변경
- 수업분류 탭 추가/수정/삭제/순서 변경
- 수업 종목/분류
- 요일
- 시작/종료 시간
- 수업명
- 학년
- 정원
- 현재원
- 장소

## 배포 후 주소

- 학부모 화면: `/`
- 관리자 화면: `/admin`
- 저장 함수 확인: `/.netlify/functions/schedule`
